# 🌐 Personal Portfolio Website

A modern and responsive personal portfolio website built to showcase my skills, projects, and web development journey.  
This portfolio highlights my work, technical abilities, and provides an easy way to connect with me.

## 🚀 Features

- Responsive design for desktop and mobile
- Smooth scrolling and modern UI
- Hero section with introduction
- About section
- Skills section
- Projects showcase
- Contact section
- Clean animations and interactive effects

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 📂 Project Purpose

This portfolio website was created to present my profile, technical skills, and projects in a professional way.  
It serves as an online resume and project showcase for recruiters, clients, and collaborators.
